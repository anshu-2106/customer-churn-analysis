# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes telecom customer churn data using SQL and Excel dashboard visualization.

## Tools Used
- SQL (MySQL)
- Microsoft Excel

## Analysis Performed
- Customer churn rate analysis
- Churn by contract type
- Churn by gender
- Churn by tenure
- KPI analysis

## Key Insights
- Overall churn rate is 26.5%
- Month-to-month contracts have highest churn
- Customers with low tenure churn more
- Gender has minimal impact on churn
